### Ghost Docker

Dockerized Ghost Blog Implementation. You can use it using the following commands:

1. up:
	Start the container first time.

2. up-non-daemon:
	Start up the container first time.

3. start:
	Start container

4. stop:
	Stop container

5. restart:
	Restart container

6. shell:
	Access container shell

7. log:
	Access log
